# Customer Data Cleaning Project

## Overview
This project focuses on cleaning and preprocessing raw customer marketing data using Python and Pandas to prepare it for business analysis.

## Dataset
- 2,240 customer records
- 29 columns
- Customer marketing dataset

## Issues Identified
- Missing income values
- Date formatting inconsistencies
- Missing total spending metric

## Cleaning Steps
- Identified null values
- Replaced missing income values using median
- Converted date column into datetime format
- Created Total Spending column
- Exported cleaned dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- VS Code
